# Projekt-Slash-Commands

Hier wachsen projektspezifische Slash-Commands für `prfh`. Jede Claude-Instanz darf
welche ergänzen (über den normalen Issue → PR-Flow, siehe `CLAUDE.md`).

Eine Datei pro Command, kleine Dateien bevorzugt → weniger Merge-Konflikte.

Noch keine Commands definiert — sie entstehen, sobald sich ein Ablauf wiederholt
(z. B. ein Command, der die Claim-Sequenz aus `CLAUDE.md` automatisiert).
